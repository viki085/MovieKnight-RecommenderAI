# AI Movie Recommendation System 

MovieKnight RecommenderAI is an AI-powered movie recommendation system that combines semantic search with large language models to deliver contextually aware movie suggestions. The system scrapes movie data from The Movie Database (TMDB), converts it into vector embeddings, and uses a RAG architecture to generate natural-language recommendations based on user queries.

Users can describe their preferences in plain language (e.g., "action movies with space theme"), and the system retrieves relevant movies from its vector database, then uses an LLM to generate personalized recommendations with explanations.
